# CAT Tools Collection

This repository houses a suite of utility tools designed specifically for CAT CMC Admins to streamline daily workflows.

## Available Tools

### 1. HTML Cleaner (CSS Remover)

**Location:** `pages/css_remover.html`

A utility to clean HTML code for translation purposes.

- **Key Features:**
  - Removes all CSS `<style>` blocks and `<link>` tags.
  - Strips all `class` and `style` attributes from HTML elements.
  - Cleans up the code to ensure it's ready for translation tools.
  - Provides a "Copy to Clipboard" feature for the cleaned output.

### 2. Streamlined Banner Generator

**Location:** `pages/streamlined_banners_generator.html`

A powerful form-based generator for creating promotional banner HTML code.

- **Key Features:**
  - Support for multiple languages (English, Spanish, Portuguese, French, German, Italian, etc.).
  - Configurable promotion types (Category vs. Site Wide).
  - Auto-generates file naming conventions tailored for CAT systems.
  - Live visual preview of the generated banner.
  - Produces ready-to-use HTML code including responsive ribbons and specific tracking classes.
